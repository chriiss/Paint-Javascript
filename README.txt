Création d'un paint 
Fonctionnalités :
Dessiner avec un crayon 
Effacer avec la gomme 
Changer de couleur (bordure, fond, ombre)
Créer des formes (cercle, rectangle, triangle, cœur vide ou plein)
Importation d'images
Clear le canvas 
Enregistrer les actions produites dans le canvas
Bonus vidéo format mp4 en fond
Langages Utilises : HTML5, CSS3, JavaScript, Jquery
Framework Utilise : Bootstrap3
